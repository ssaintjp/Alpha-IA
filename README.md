## 🎯 Objetivo do Projeto

O projeto **Alpha IA** foi desenvolvido para demonstrar como o design pode transmitir autoridade, inovação e tecnologia de ponta. O desafio foi criar uma interface que não apenas apresentasse produtos de IA, mas que fizesse o usuário *sentir* a tecnologia através de interações fluidas e estética premium.

---

## 🎨 Design Philosophy (Filosofia de Design)

Para este projeto, adotei uma estética **Dark Premium** combinada com elementos de **Glassmorphism**.

### 🛠️ Identidade Visual
*   **Paleta de Cores**: Baseada em um preto profundo (`#000000`) para evocar sofisticação, com acentos em gradientes vibrantes de **Azul Royal** e **Roxo Elétrico**.
*   **Tipografia**: Utilização da fonte **Poppins** (Sans-serif geométrica) para garantir legibilidade e um ar de modernidade técnica.
*   **Hierarquia**: Uso estratégico de contrastes e sombras internas para guiar o olhar do usuário através dos produtos e cases de sucesso.

### ✨ Destaques de UI/UX

1.  **Custom Cursor (Cursor Interativo)**:
    *   Um cursor personalizado que utiliza `mix-blend-mode: difference`, adaptando-se às cores do fundo.
    *   **Micro-interação**: O cursor expande e ganha opacidade ao pairar sobre elementos clicáveis, fornecendo feedback tátil visual.

2.  **Stacked Cards Experience (Cases de Sucesso)**:
    *   Uma seção de cases que utiliza profundidade e sobreposição.
    *   **UX Detail**: Scroll horizontal com barra de progresso customizada, criando uma navegação intuitiva em dispositivos móveis e desktop.

3.  **Glassmorphic Navbar**:
    *   O cabeçalho inicia transparente e, ao rolar a página, transiciona suavemente para um estado com `backdrop-filter: blur`, mantendo a elegância sem sacrificar a funcionalidade.

4.  **Micro-delights**:
    *   Efeito de digitação (Typewriter) no Hero para dinamismo.
    *   Animações de pulso em CTAs críticos.
    *   Sistema de feedback com confetes ao interagir com botões principais, aumentando o engajamento emocional.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído "do zero" (Vanilla), demonstrando domínio sobre as bases do desenvolvimento web:

*   **HTML5 Semântico**: Estrutura focada em acessibilidade e SEO.
*   **CSS3 Avançado**: Uso extensivo de Variáveis (Custom Properties), Flexbox, Grid e Animações `@keyframes`.
*   **JavaScript (ES6+)**: Lógica personalizada para o cursor, scroll progress, chat interativo e modais, sem dependência de bibliotecas externas pesadas.

---

## 📱 Responsividade

O design foi concebido com uma abordagem **Mobile-First**, garantindo que a experiência premium se mantenha em telas menores:
*   Menu lateral (Drawer) intuitivo para dispositivos móveis.
*   Ajuste inteligente de grades (Grid) e tamanhos de fonte.
*   Imagens e vídeos otimizados para carregamento rápido.
